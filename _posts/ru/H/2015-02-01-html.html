HTML Events
body Events
onload Скрипт выполняется, когда документ загружается
onunload Скрипт выполняется, когда документ выгружается
Form Events
onblur Скрипт выполняется, когда элемент теряет фокус
onchange Скрипт выполняется, когда элемент изменяется
onfocus Скрипт выполняется, когда элемент попадает в фокус
onreset Скрипт выполняется, когда форма сбрасывается
onselect Скрипт выполняется, когда элемент выбран
onsubmit Скрипт выполняется, когда форма отправлена
Image Events
onabort Скрипт выполняется, когда загрузка изображения прерывается
Mouse Events
onclick Скрипт выполняется, когда нажимается мышь
ondblclick Скрипт выполняется при двойном клике мыши
onmousedown Скрипт выполняется, когда клавиша мышки нажата
onmousemove Скрипт выполняется, когда указатель мышки сдвинулся
onmouseout Скрипт выполняется, когда указатель мыши перемещается из одного элемента
onmouseover Скрипт выполняется, когда указатель мышки проходит по элементу
onmouseup Скрипт выполняется когда клавиша мышки отпущена
